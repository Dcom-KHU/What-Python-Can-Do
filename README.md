## 실행법
Windows
```bash
$ python -m venv venv
$ call venv/Scripts/activate
$ pip install -r requirement.txt
```

Linux
```bash
$ python -m venv venv
$ source venv/Scripts/activate
$ pip install -r requirements.txt
```
